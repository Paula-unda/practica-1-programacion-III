## Ejercicio 1: Node.js 

### Descripción
Aplicación que solicita el nombre de un animal favorito mediante un formulario y, usando Node.js con Express, recibe ese dato y muestra el resultado en una segunda página.

Se utilizan dos vistas:
- Formulario de entrada
- Página de resultado


### Requisitos previos
- Node.js instalado (versión 18 o superior)

### Instalación y ejecución

```bash
# Navegar a la carpeta del ejercicio 1
cd ejercicio1

# Instalar dependencias
npm install express

# Ejecutar el servidor
node server.js
````

## Ejercicio 2: TypeScript 

### Descripción
Programa en TypeScript que utiliza enumerados (enums) para clasificar géneros de películas y países de origen. Los valores se muestran por consola de forma estructurada, demostrando la seguridad de tipos que ofrece TypeScript.

### Requisitos previos
- Node.js instalado (versión 18 o superior)
- TypeScript instalado globalmente:

```bash
npm install -g typescript
````

### Instalación y ejecución

```bash
# Navegar a la carpeta del ejercicio 2
cd ejercicio2

# Compilar TypeScript a JavaScript
tsc

# Ejecutar el archivo compilado
node dist/peliculas.js
```
